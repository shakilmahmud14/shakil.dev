jQuery Background Slider To Resize Images to Background Window

jQuery background slider shows background slide show using jQuery plugin.

It uses parameters to control the speed and interval between two slides.

Images are stretched to the size of the background window.

It shows the fixed background on page scroll.

On Page scroll the content goes up and down over the fixed background.
